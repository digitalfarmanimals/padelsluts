# 🎾 Padelsluts

Americano padel tournaments — add any number of players, set the points per match
and number of rounds, and it randomly re-pairs everyone each round. Final table
ranks players by points (or wins) and crowns one overall winner.

- **Any number of players** — makes as many courts as fit (4 per court), rotates byes fairly
- **Set points per match** — 16 / 21 / 24 / 32 or any number; scores auto-complete
- **Pick the rounds** — random partners + opponents every round, reshuffle anytime
- **Ranking toggle** — by total points (default) or by wins
- **Installable & offline** — add to your home screen, works with no signal at the court

Single static web app — no build, no backend. Just open `index.html`.

Live: https://digitalfarmanimals.github.io/padelsluts/
